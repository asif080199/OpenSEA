\subsection{Detailed Description}
The \hyperlink{classosea_1_1ofreq_1_1_hydro_manager}{Hydro\-Manager} class contains the complete set of hydrodynamic data associated with a single body. 

The most important aspect are all the variations possible for hydrodynamic data. The data can vary with wave frequency, wave direction, and wave amplitude. The \hyperlink{classosea_1_1ofreq_1_1_hydro_manager}{Hydro\-Manager} has methods for interpolating amongst all that data to create a single output for a requested set of conditions (some combination of wave frequency, wave direction, and wave amplitude.)

Note that interpolation for wave direction does have a limitation\-: it will only permit interpolation with wave directions that are within 0.\-785 rad (45 deg) of the available data. This is due to the physics limitations. Any data more than 45 deg away from the desired direction really is not relevant.
