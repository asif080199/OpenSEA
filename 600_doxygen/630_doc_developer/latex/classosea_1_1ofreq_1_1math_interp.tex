\subsection{Detailed Description}
The \hyperlink{classosea_1_1ofreq_1_1math_interp}{math\-Interp} class contains all math functions used for interpolation and extrapolation of data. 

The class is designed to be used for internal functions in other classes. Many of the functions are designed as generic templates to allow large compatibility with other object types.
