The main function that runs ofreq program. 

The main function that runs ofreq program. It proceeds through in several stages. This briefly outlines them. 0. Initialize a few variables.
\begin{DoxyEnumerate}
\item Read in input files.
\item Interpret / parse input files and use them to build the program objects.
\item Use the motion model to convert input objects into matrix force objects. The particular motion model used changes with each Body object.
\item Setup the motion solver. Feed all the data in.
\item Set the operating frequency and use the motion solver to solve equations of motion.
\item Store results in Solution object.
\item Repeat steps 4 through 6 for each wave direction and wave frequency.
\item Use the results to calculate derived outputs.
\item Write the calculated outputs to output files. 
\end{DoxyEnumerate}
