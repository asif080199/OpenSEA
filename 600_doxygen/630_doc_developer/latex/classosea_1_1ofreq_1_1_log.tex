\subsection{Detailed Description}
The \hyperlink{classosea_1_1ofreq_1_1_log}{Log} class is used to create output logs. 

The \hyperlink{classosea_1_1ofreq_1_1_log}{Log} class creates output logs. You can use this class to write outputs to a log file, to the screen, or to both. You also have the options to set output format. You can write output with an actual date and time assigned, or with just a direct message.
