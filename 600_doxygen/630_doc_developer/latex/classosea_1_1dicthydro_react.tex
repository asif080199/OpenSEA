\subsection{Detailed Description}
Prototype class declaration for pointer to hydro\-Data object. The dict\-Hydro\-React class is a template class for the dictionaries that read reactive forces.

All of these input files have exactly the same format. They are just associated with a different order of derivative. A single static variable changes for each of the children classes. Everything else remains the same.

The dict\-Hydro\-React class is itself derived from the \hyperlink{classosea_1_1_dictionary}{Dictionary} class. The \hyperlink{classosea_1_1_dictionary}{Dictionary} class defines most of the behavior. The only thing added to this class is the behavior for how to handle the individual keywords specific to this input file. In the case of the dict\-Hydro\-React class, the same keywords are used in all related input files, hence why a generic class can be used with inheritence.
