ObjectGroupGroupings of object definitions captured from an input file. It is a data container to hold the segmented input file for interpretation. The container contains three things: 1.) Object class name (as specified by input file) 2.) Vector of keyword names 3.) Vector of keyword values. Each entry in the vector of values is also a vector. This allows the definition of lists. A list will be as long as it needs to be for specification of all values in the list. The index of the value is specified by its position in the vector list. The value is the entry
ParserTakes an input segment of strings and segments that segment. It strips out comments. It recognizes quotation marks and groups those segments together. Parser finally returns a series of ObjectGroup objects. Each ObjectGroup contains the classname and a list of keyword value pairs
