Detailed Description

This class write all outputs to files. All output data is based on the attached OutputsBody object. To use the filewriter object, follow this sequence of steps: 1.) Create object. 2.) Set OutputsBody object associated with the file. 3.) Set the filesystem path to the root directory for the current run of ofreq. 4.) Run the clearFiles() function, which will clear out any pre-existing files. 5.) Run the writeFile function for the specified file that you want to write out.

Note that the OutputsBody object also provides the information on the current wave direction. And the FileWriter changes its directory to write to depending on the current wave direction. So the OutputsBody object must be updated before writing a new wave direction.
