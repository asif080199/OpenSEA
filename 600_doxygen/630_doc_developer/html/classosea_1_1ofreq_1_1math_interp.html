Detailed Description

The mathInterp class contains all math functions used for interpolation and extrapolation of data.

The class is designed to be used for internal functions in other classes. Many of the functions are designed as generic templates to allow large compatibility with other object types.

A linear interpolation function.

This function will perform linear interpolation or extrapolation, based on two data points. It is a template function and can take almost any data type. The independent variables (the x variables) in the function must be double data type, but the dependent variables (the y variables) can be almost any data types, assuming the data type supports certain operators.

Parameters
xDouble, variable passed by value. This is the x value for the output that you want. The result of the interpolation will be based on this x value.
x1Double, variable passed by value. This is the first x value for the data that you have.
x2Double, variable passed by value. This is the second x value for the data that you have.
y1Template variable, passed by value. This is the first y value for the data that you have. This y value corresponds to the x1 value given earlier. The object type passed to this parameter must support the following operations: (+, -, *)
y2Template variable, passed by value. This is the second y value for the data that you have. This y value corresponds to the x2 value given earlier. The object type passed to this parameter must support the following operations: (+, -, *)
Returns
Returned type is the template type. Variable is passed by value.
