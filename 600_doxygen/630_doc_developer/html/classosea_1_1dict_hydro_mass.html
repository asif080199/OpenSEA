Detailed Description

The dictHydroDamp class interprets the input file hydrodamp.out

The hydrodamp.out input file is a set of reactive forces, associted with a first order derivative. All of these input files have exactly the same format. They are just associated with a different order of derivative. A single static variable changes for each of the children classes. Everything else remains the same. The dictHydroReact class contains all the function definitions for how to interpret the input files.

The only thing added in this class is the order of derivative associated with the input file. This is set through a single static variable. ORD = 1 for this class.
