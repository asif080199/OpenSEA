Detailed Description

The SpecBretschneider class. This defines a Bretschneider wave spectrum. Also called an ITTC wave spectrum. Assumes fully developed sea. The user defines two parameters: Significant wave height and mean period. The class allows definition of the mean period through one of three options: mean period, zero cross period, or peak period.
